
Trabalho para a disciplina de comunicação e rdes da UFABC.
O script tarefa1.py salva o visualizazao.jpg numa pasta images/, o script tarefa2.py também salva as imagens das tabelas de métricas.

No tarefa1.py é possível usar um parâmetro para renderizar o grafo os nomes dos aeroporto na forma:
> python tarefa1.py airportName 

No tarefa2.py várias étricas são calculadas e são visualizadas uma por vez, ao fechar a janela de uma, outra se abre até última métrica ser visualizada.
